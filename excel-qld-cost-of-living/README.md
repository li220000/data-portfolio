# Queensland Cost of Living Dashboard

## Project Overview
This project analyses cost-of-living trends in Queensland using publicly available data. The goal is to identify key drivers of living costs and provide insights through an interactive Excel dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- XLOOKUP
- Data Cleaning
- Data Visualisation (Charts & Dashboard)

## Data Sources
- Queensland / Australian public datasets (e.g. CPI, rent, income)

## Data Preparation
- Cleaned and standardised raw datasets
- Handled missing values
- Merged multiple datasets using XLOOKUP
- Created structured tables for analysis

## Key Analysis
- Cost of living trends over time (CPI / rent / income)
- Comparison across categories (housing, food, transport)
- Identification of periods with significant cost increases

## Key Insights
- Housing costs are the main driver of cost-of-living increases
- Significant rise observed post-2021
- Income growth does not fully match cost increases

## Dashboard Features
- Interactive filters (e.g. year / category)
- Trend charts for key indicators
- KPI summary (cost growth, category comparison)
