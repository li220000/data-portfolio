# Queensland Cost of Living Analysis (Excel Dashboard)

## Project Overview
This project analyses cost-of-living trends in Queensland/Brisbane using publicly available data. The goal is to identify key drivers of living costs and assess how affordability has changed over time through an interactive Excel dashboard.

## Time Scope
This project compares 2023 and 2025 data to examine how cost-of-living pressures have changed in the post-COVID period.

## Analytical Framework
Cost of living is analysed across key categories aligned with CPI components:
- Housing (rent and housing costs)
- Food
- Transport
- Clothing

Income is included to assess affordability and cost pressure.

## Tools Used
- Microsoft Excel
- Pivot Tables
- XLOOKUP
- Data Cleaning
- Data Visualisation (Charts & Dashboard)

## Data Sources
- Public datasets (e.g. CPI, rent, wage index)

## Data Preparation
- Cleaned and standardised raw datasets
- Handled missing values
- Merged multiple datasets using XLOOKUP
- Structured data into a unified analytical format

## Key Analysis
- Comparison of cost-of-living indicators between 2023 and 2025
- Category-level analysis (housing, food, transport, clothing)
- Comparison of cost growth vs income growth
- Identification of key drivers of cost increases

## Key Insights
- Housing costs showed the strongest increase and are the primary driver of rising living costs
- Food and transport costs also increased, but at a slower pace than housing
- Income growth did not keep pace with rising costs, indicating increased affordability pressure

## Dashboard Features
- Interactive filters (e.g. category selection)
- KPI summary (cost growth vs income growth)
- Trend charts for key indicators
- Category comparison visualisations
